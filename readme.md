# Symfony Sepa Bundle
Direct Debit to XML batch bundle.

## Development
While developing in `composer.json` if you checkout the bundle too:
```
"repositories": [
    {
        "type": "path",
        "url": "../../symfony-sepa"
    }
],
```

Then `composer update lensmedia/symfony-sepa` to create the new link.

Don't commit these changes.
